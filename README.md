# Flight Reservation System

## Overview
A console-based Java application that allows users to search flights, book them, and view reservations. Data is stored in-memory during runtime.

## Features
- Search flights by source, destination, and date.
- Book flights with seat availability validation.
- View all reservations.
- Unique reservation IDs generated randomly for realism.
- Unit tests using JUnit to validate booking and search functionality.

## Technologies
- Java 21
- Maven 3.9.11
- JUnit 5
- In-memory data structures (ArrayList)
