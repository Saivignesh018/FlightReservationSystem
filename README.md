A console-based Java application that allows users to search flights, book them, and view reservations. All data is stored in-memory during runtime.

## Features
- Search flights by **source**, **destination**, and **date**.
- Book flights with **seat availability validation**.
- View all reservations.
- **Unique 5-digit reservation IDs** generated randomly to mimic real airline systems.
- Unit tests using **JUnit 5** to validate booking and search functionality.

## Technologies
- **Java 21**
- **Maven 3.9.11**
- **JUnit 5**
- **In-memory data structures** (`ArrayList`)
